# 🏀 Basketball Analytics: Performance & Statistical Modeling

Este repositorio contiene un análisis profundo de datos de la NBA, integrando técnicas de **Data Science** para evaluar el rendimiento de jugadores y la eficiencia de los equipos. El objetivo es transformar estadísticas crudas en insights accionables mediante visualizaciones avanzadas y modelos predictivos.

## 🚀 Overview

El proyecto surge de la curiosidad por entender los patrones de juego modernos (como la revolución del triple) y cómo impactan en la probabilidad de victoria. Utilizo Python para el procesamiento de grandes volúmenes de datos históricos y actuales.

## 🛠️ Tech Stack

* **Lenguaje:** Python 3.x
* **Análisis de Datos:** Pandas, NumPy
* **Visualización:** Matplotlib, Seaborn, Plotly (para gráficos interactivos)
* **Entorno:** Google Colab / Jupyter Notebooks
* **Data Sourcing:** API de NBA Stats / BeautifulSoup (Web Scraping)

## 📊 Key Features & Analysis

* **Shot Selection Analysis:** Visualización de mapas de calor (*shot charts*) para identificar zonas de alta eficiencia.
* **Player Efficiency Rating (PER):** Cálculo y comparación de métricas avanzadas frente a estadísticas tradicionales.
* **Correlation Studies:** Análisis de la relación entre el ritmo de juego (*Pace*) y los puntos por posesión.
* **Team Clustering:** Agrupación de equipos según su estilo de juego defensivo/ofensivo mediante algoritmos de ML.
